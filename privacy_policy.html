Property Value Tool – Privacy Policy

Last updated: 31 March 2026

Property Value Tool is a browser extension that reads information from property listing pages you choose to open, in order to provide buyer-side analysis such as price context, comparable evidence, and area insights.

What the extension accesses

  • Property listing information visible on supported pages, such as price, address, property type, bedrooms, bathrooms, tenure, description, and listing URL.
  • Optional user inputs you provide, such as search radius or fallback property details.
  • Locally cached results to make repeat use on the same listing faster.

How data is used

  • To generate buyer-side property analysis and comparable evidence.
  • To improve performance by storing limited local cache data in the browser.
  • To request supporting data from configured backend services or public data sources when needed.

What is not sold

We do not sell your personal information.

Data storage

The extension may store limited data in your browser, such as the last analysed listing and related cached results. If backend services are used, property-related data may be processed on those services in order to return analysis results.

Third-party services

The extension may call external services to fetch public property, mapping, or area information. These services may process technical request data such as IP address, browser metadata, and requested listing details in accordance with their own policies.

Your choices

  • You can remove the extension at any time.
  • You can clear locally stored extension data through your browser settings.

Contact

For privacy questions, use the support/contact link listed on the extension page.
